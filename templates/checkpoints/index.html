Sistema de Puntos de Fichaje

Descripción del Sistema

El sistema de puntos de fichaje permite controlar la entrada y salida de los empleados mediante dispositivos móviles o tablets asociados a cada empresa.

Características principales:

  • Creación de puntos de fichaje para cada empresa
  • Autenticación de empleados mediante PIN
  • Registro automático de horas trabajadas
  • Ajustes automáticos según horas de contrato
  • Registro de incidencias
  • Exportación de informes con firma digital

Acceso para Empleados

Para que los empleados accedan al sistema de fichaje, deben usar la URL específica del punto de fichaje:

https://su-dominio.com/fichajes/login

Cada punto de fichaje tiene sus propias credenciales de acceso que deben ser proporcionadas a los responsables de administrar el dispositivo de fichaje.
